GUIDE D'INSTALLATION

PPI Analyser

Installation pas à pas sur Windows.

1 · Python 2 · Podman 3 · Lancer le script 4 · Clé API 5 · Vérification

01 Installer Python

🪟
Ouvrir le Microsoft Store

Cherchez python dans la barre de recherche du Store. Sélectionnez Python 3.13 (Python Software Foundation).

📥
Cliquer sur « Install »

L'installation est automatique, aucune configuration requise. Python sera accessible depuis le terminal une fois l'installation terminée.

✔ Vérifiez l'installation : ouvre un terminal (Win + R → cmd) et tapez python --version. Tu dois voir Python 3.13.x.

02 Installer Podman

🌐
Aller sur podman.io

Cliquez sur le bouton Download en haut de la page.

⬇️
Télécharger « Podman CLI for Windows »

Sélectionnez Podman CLI for Windows (v5.8.3 ou plus récent). Ne pas choisir Podman Desktop — le CLI suffit.

⚙️
Lancer l'installeur — choisir Hyper-V

Dans l'écran de setup, sélectionnez Windows Hyper-V comme fournisseur de virtualisation, puis cliquez Install.

⚠ Un redémarrage peut être demandé après l'installation de Hyper-V. Redémarre et relance l'installeur si nécessaire.

03 Lancer le script d'installation

📁
Placer les fichiers dans un même dossier

Le dossier doit contenir LANCER_PPI.bat, docker-compose.yml et setup_ppi.py côte à côte.

🖱️
Clic droit sur LANCER_PPI.bat → « Exécuter en tant qu'administrateur »

Les droits administrateur sont nécessaires pour que Podman puisse créer et démarrer sa machine virtuelle.

🛡️
Accepter l'avertissement Windows

Windows affiche une alerte « éditeur inconnu ». Cliquez sur Run pour continuer — c'est normal pour un fichier non signé.

ℹ Une fenêtre GUI s'ouvre. Elle gère automatiquement : initialisation de la VM Podman (~700 MB au premier lancement), téléchargement de l'image Docker, démarrage du conteneur.

04 Renseigner la clé API Mistral

⚙️
Ouvrir le panneau Admin

Pour pouvoir utiliser l'outil avec Mistral, il faut saisir votre clé API. Pour ce faire, cliquez sur le bouton « Admin » en bas à droite de l'interface.

📋
Sélectionner MISTRAL_API_KEY

Dans la liste déroulante à gauche du panneau Admin, choisissez MISTRAL_API_KEY.

🔑
Saisir la clé

Saisissez votre clé API Mistral dans l'espace prévu à droite, puis enregistrez. Si vous redémarrez l'outil, il faudra la saisir de nouveau. Afin de faciliter cela, il est possible d'enregistrer la clé dans le gestionnaire de mots de passe du navigateur

05 Vérifier que l'application est prête

▶️
Cliquer sur « Démarrer »

Le script exécute les étapes dans l'ordre et affiche leur statut en temps réel dans le journal de droite. La VM Podman peut prendre 1 à 2 minutes au premier lancement.

🌐
Le navigateur s'ouvre automatiquement

Une fois l'application prête, le bouton devient « Ouvrir PPI Analyser » et http://localhost:8000 s'ouvre dans votre navigateur.

⚠ Les modèles NLP (Stanza) peuvent prendre 1 à 2 minutes supplémentaires à charger après le démarrage du conteneur. Si la page ne répond pas immédiatement, patientez et actualisez.
🛑
Fermer = arrêter le serveur

Fermer la fenêtre du script arrête automatiquement le conteneur Podman. Aucune manipulation manuelle n'est nécessaire.

✔ Récapitulatif

  • ✔ Python 3.13 installé depuis le Microsoft Store
  • ✔ Podman CLI installé avec le backend Hyper-V
  • ✔ LANCER_PPI.bat et setup_ppi.py dans le même dossier
  • ✔ Script lancé en administrateur
  • ✔ Clé API Mistral renseignée
  • ✔ Application accessible sur http://localhost:5000
